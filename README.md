# Coin Base
> Example demo coin base with block chain.

## Installation

OS X & Linux & Windows:

```sh
$ cd ROOT-FOLDER
$ npm install
```

## Run Coin Base

> Server
```sh
$ cd ROOT-FOLDER
$ npm start
```
> Client
```sh
$ cd ROOT-FOLDE
Open file index.html
```
